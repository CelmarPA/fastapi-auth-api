# Postman Collection

Este diretório contém a coleção Postman para testar a API.

## Como importar

1. Abra o Postman
2. Clique em "Import"
3. Selecione o arquivo `Auth API.postman_collection.json`
4. A API estará pronta para uso

## Estrutura da Collection

- Auth
  - Register
  - Login
  - Refresh Token
  - Logout
  - Request Password Reset
  - Request Password
  - Me
  - Send Email Verification
  - Verify Email
- Users
  - List Users
  - Get User by ID
  - Update User
  - Delete User
  - Disable User
  - Enable User
  - Admin Dashboard
- Products
  - List Products
  - Create Product
  - Get Product by ID
  - Update Product
  - Delete Product
